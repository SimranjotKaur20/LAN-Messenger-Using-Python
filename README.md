# LAN Messenger Using Python

A LAN-based messaging application developed using Python and TCP socket programming. The application allows devices connected to the same LAN, Wi-Fi, or mobile hotspot network to exchange messages in real time.

## Features
- Client-Server Architecture
- Real-time Messaging
- TCP Socket Communication
- Works on LAN and Mobile Hotspot
- Easy to set up and use

## Technologies Used
- Python
- Socket Module
- TCP Protocol
- VS Code

## Project Structure

LAN-Messenger-Using-Python/
│
├── server.py
├── client.py
└── README.md

## How to Run

### Server
```bash
python server.py
```

### Client
```bash
python client.py
```

Update the server IP address in `client.py` before running the client.

## Network Requirements
- Both devices must be connected to the same LAN, Wi-Fi, or mobile hotspot.
- The server must be running before the client connects.
